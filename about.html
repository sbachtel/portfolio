Steve Bachtel is a freelance photographer and web developer. He splits his time between Seattle, Washington and Kansas City, Missouri. Steve has worked in the photo industry for the last 7 years, and has worked as a web developer for the last 2 years.

His skills include CSS3, LESS for CSS3, Git for SCM, Javascript, HTML5, AngularJS, Bootstrap. Development tools include Sublime, IntelliJ and various browser development consoles. Graphic development tools include Adobe PhotoShop, Lightroom and Illustrator.
